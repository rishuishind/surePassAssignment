Demo Test Page

This page demonstrates the CookieGuard consent banner embed. The banner above (if configured) is loaded from the CookieGuard backend.

How This Works

This page has the CookieGuard embed script installed. When you visit this page:

1. Embed Script Loads

The <script> tag in the head loads banner.js from the CookieGuard backend. It reads the data-site-id to know which banner config to fetch.

2. Banner Config Fetched

The script calls GET /api/public/banner/:siteId to retrieve the latest active banner configuration saved in the dashboard.

3. Banner Rendered

The consent banner is injected into the page DOM with your custom colors, text, position, and category options.

4. Consent Recorded

When a visitor clicks Accept, Reject, or saves preferences, a consent event is sent to POST /api/public/consent and appears in your Consent Log.

Setup Steps

1. Open the CookieGuard dashboard at http://localhost:3000

2. Add a site and scan it

3. Go to Banner Builder → configure and save a banner

4. Copy the site ID from the embed snippet and update the data-site-id in this file

5. Open this page — the banner should appear

6. Click Accept/Reject/Preferences and check the Consent Log in the dashboard
